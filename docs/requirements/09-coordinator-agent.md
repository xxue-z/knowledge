# 需求文档：协调 Agent

## 1. 需求概述

协调 Agent 负责复合任务分解、工作流执行、跨库查询协调和结果聚合，是实现多 Agent 协作的核心组件。

## 3. 跨库查询流程示例

### 场景：张三本月考勤信息和迟到扣款

| 步骤 | 执行 Agent | 操作 | 来源 |
|------|------------|------|------|
| 1 | 路由 Agent | 提取用户身份，调用权限 Agent | README.md 第6节 |
| 2 | 权限 Agent | 校验查询权限 | README.md 第6节 |
| 3 | 协调 Agent | 分解任务为：查询考勤 + 查询制度 | README.md 第6节 |
| 4 | 轻库 Agent | 查询张三打卡记录（带 dept_id 过滤） | README.md 第6节 |
| 5 | Wiki Agent | 提供考勤制度文本 | README.md 第6节 |
| 6 | 协调 Agent | 计算扣款，返回结果并标注来源 | README.md 第6节 |
