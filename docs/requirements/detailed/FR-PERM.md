# FR-PERM: 权限Agent模块需求

## FR-PERM-001: RBAC角色鉴权
- **描述**: 使用Casbin进行角色权限校验
- **参与者**: 系统内部
- **前置条件**: 用户发起操作请求
- **主成功场景**:
  1. 用户请求资源
  2. 系统校验角色权限
  3. 允许或拒绝请求
- **拟合标准**: 权限校验正确，响应时间<100ms
- **来源**: PRD用户故事#10
- **KANO类别**: 基本型
- **优先级**: P0
- **状态**: ✅ 已完成

## FR-PERM-003: 动态策略热更新
- **描述**: Casbin策略更改后立即生效，支持通过API端点触发重载
- **参与者**: 管理员
- **前置条件**: 修改权限策略
- **主成功场景**:
  1. 管理员修改策略
  2. 调用 POST /api/admin/policies/reload 触发重载
  3. 策略自动重新加载，新策略立即生效
- **拟合标准**: 策略更新后5秒内生效
- **来源**: PRD用户故事#23
- **KANO类别**: 期望型
- **优先级**: P2
- **状态**: ✅ 已完成
