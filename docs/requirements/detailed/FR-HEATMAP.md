# FR-HEATMAP: 热力图模块需求

## FR-HEATMAP-001: 检索事件埋点
- **描述**: 系统自动记录用户检索行为，用于热度统计分析
- **参与者**: 系统内部（检索服务层）
- **前置条件**: 检索事件触发
- **主成功场景**:
  1. 用户发起检索请求
  2. 系统异步记录检索事件到PostgreSQL和Redis
  3. 检索结果正常返回给用户（埋点不影响主流程）
- **扩展场景**:
  - Redis不可用：降级为仅写入PostgreSQL
  - PostgreSQL写入失败：记录日志，不影响检索流程
- **拟合标准**: 埋点记录成功率≥99%，埋点对检索响应时间影响≤5ms
- **来源**: 2026-05-23-heatmap-design.md
- **KANO类别**: 期望型
- **优先级**: P1
- **状态**: ⚠️ 待实现
