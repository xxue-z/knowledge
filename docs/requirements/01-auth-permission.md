# 需求文档：身份认证与权限管理

## 1. 需求概述

基于项目 README.md，身份认证与权限管理是系统的核心安全层，确保：
- 用户身份验证（登录/登出）
- 细粒度数据权限分级
- 角色级访问控制

---

## 3. 非功能需求

| 需求编号 | 需求描述 | 来源 |
|----------|----------|------|
| AUTH-NFR-001 | 权限校验响应时间 < 100ms | 性能需求 |
| AUTH-NFR-002 | 支持1000+并发用户认证 | 性能需求 |
| AUTH-NFR-003 | 所有API调用必须携带token | 安全需求 |
| AUTH-NFR-004 | 权限策略变更记录审计日志 | 合规需求 |

## 4. 数据权限分级矩阵

| 数据类型 | 普通员工 | 部门经理 | HR | 管理员 |
|----------|----------|----------|-----|--------|
| 个人档案 | 仅自己 | 本部门+下级 | 全公司(脱敏) | 全部 |
| 个人会话 | 仅自己 | 本部门+下级 | - | 全部 |
| Wiki公开文档 | 可读 | 可读 | 可读 | 可管理 |
| Wiki内部文档 | 部门内 | 本部门+下级 | 全公司 | 可管理 |
| 敏感Wiki文档 | 受限 | 受限 | 受限 | 可管理 |
