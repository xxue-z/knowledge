# 需求文档：知识导航系统

## 1. 需求概述

知识导航系统提供多层级树形分类结构，支持人工维护与 Agent 自动归纳，实现企业知识的有序组织和高效发现。

## 3. 数据模型需求

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | UUID | 主键 | 节点唯一标识 |
| name | String | max=200 | 节点名称 |
| parent_id | UUID | 外键 | 父节点ID |
| path | String | ltree | 物化路径 |
| icon | String | 可选 | 图标名称 |
| description | String | 可选 | 描述信息 |
| sort_order | Integer | 默认=0 | 排序顺序 |
| visibility_roles | JSON | 默认=[] | 可见角色列表 |
| created_at | DateTime | - | 创建时间 |
| updated_at | DateTime | - | 更新时间 |
